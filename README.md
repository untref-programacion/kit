# Kit

## setup

### Prerequisitos

Es necesario tener instalado [Node.js](https://nodejs.org/). Se recomiendo usar NVM para manejar las versiones de node.

- Linux, macOS: <https://github.com/nvm-sh/nvm>
- Windows: <https://github.com/coreybutler/nvm-windows>

De todos modos con tener instalado Node en su versión 20 o superior es suficiente (idealmente la última versión LTS).

### Sólo la primera vez

Clonamos el repo e instalamos las dependencias de Node.

```shell
git clone git@github.com:untref-programacion/kit.git
```

```shell
cd kit
npm install
```

### Iniciando el servidor local

Cada vez que vayamos a trabajar localmente, es útil inicial el servidor que nos permitirá previsualizar como quedará el contenido

```shell
npm run start
```

### Visual Studio Code

Como IDE se recomienda utilizar Visual Studio Code. En este mismo proyecto, hay algunas configuraciones compartidas que además incluyen extensiones recomendadas que el editor va a ofrecerles instalar la primera vez que carguen el proyecto.
